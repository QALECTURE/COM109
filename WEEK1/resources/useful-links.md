# Useful Links

## Web Development Documentation

- [MDN Web Docs](https://developer.mozilla.org/)  
  A professional reference for HTML, CSS, JavaScript and browser APIs.

- [W3Schools](https://www.w3schools.com/)  
  Beginner-friendly tutorials and examples for web development.

## Source Control and GitHub

- [GitHub](https://github.com/)  
  A platform for hosting, sharing and collaborating on code.

- [GitHub Education](https://education.github.com/)  
  Student resources, benefits and learning materials from GitHub.

## Practice and Learning

- [freeCodeCamp](https://www.freecodecamp.org/)  
  Free coding lessons and practice projects.

- [CodePen](https://codepen.io/)  
  A browser-based tool for experimenting with HTML, CSS and JavaScript.

- [JSFiddle](https://jsfiddle.net/)  
  Another browser-based tool for testing small web examples.

## Browser Tools

- [Chrome DevTools Documentation](https://developer.chrome.com/docs/devtools/)  
  Official documentation for inspecting, debugging and testing webpages in Chrome.
