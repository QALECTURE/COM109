# Activity 02: HTML Tag Hunt

## Purpose

Students use browser DevTools to inspect a real website and identify common HTML tags.

## Time

15 minutes

## Suggested Websites

Choose one:

- BBC
- Amazon
- GitHub
- Netflix homepage
- University website
- Any well-known public website

## Instructions

1. Open a website in your browser.
2. Right-click the page and choose **Inspect**.
3. Use the Elements panel to search for common HTML tags.
4. Complete the worksheet table below.

## Tags to Find

- `h1`
- `p`
- `img`
- `a`
- `button`
- `form`

## Follow-Up Questions

- Was the HTML easy to read?
- Did the page have more elements than you expected?
- Why might large websites have complicated HTML?
- Which tag appeared most often?

## Lecturer Note

Students may not always find every tag quickly. That is fine. The goal is to show that real websites are built from the same basic building blocks they are learning.
