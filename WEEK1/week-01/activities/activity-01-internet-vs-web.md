# Activity 01: Internet vs Web

## Time

10 minutes

## Purpose

Help students understand that the Internet and the World Wide Web are related, but not the same thing.

## Instructions

Work in pairs. Classify each item as:

- **Web-based**
- **Internet service, but not mainly the Web**
- **Can be both, depending on how it is used**

## Items to Classify

| Item | Classification | Why? |
| --- | --- | --- |
| WhatsApp | Can be both | The mobile app uses the Internet. WhatsApp Web uses the Web in a browser. |
| Gmail | Can be both | Gmail in a browser is Web-based. Email itself is an Internet service. |
| Google Search | Web-based | It is mainly accessed through webpages in a browser. |
| FTP | Internet service, but not mainly the Web | FTP transfers files using a different protocol from HTTP/HTTPS. |
| Netflix | Can be both | The website is Web-based. Apps also use the Internet to stream video. |
| SSH | Internet service, but not mainly the Web | SSH lets users securely connect to remote machines through a terminal. |
| YouTube | Can be both | The website is Web-based. The mobile and TV apps use the Internet too. |

## Discussion Questions

- Which examples surprised you?
- Why do people often use "Internet" and "Web" as if they mean the same thing?
- What do browsers mainly access?
- Which services do you use every day that depend on the Internet?

## Lecturer Wrap-Up

The Internet is the global connection system. The Web is one major service that runs on it using browsers, URLs and HTTP/HTTPS. Many modern products, such as Netflix, Gmail and YouTube, can exist as both websites and apps.
